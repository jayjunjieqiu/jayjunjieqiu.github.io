# Repository Instructions

## Image Assets

- When adding or replacing image assets, remind the user to decide whether the image should be converted to an appropriately sized WebP for its intended display size.
- Do not commit large original photos when a smaller WebP is sufficient for the website.
- Prefer WebP assets sized close to their rendered dimensions, with modest high-DPI headroom.
- Keep original images outside the repository unless the user explicitly asks to preserve them in source control.
